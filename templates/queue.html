How to Use the Queue

  1. Enter a value in the text box to add to the queue.
  2. Click 'Enqueue' to add the element to the end of the queue.
  3. Click 'Dequeue' to remove the element from the front of the queue.

Remember: In a queue, the first element added is the first one to be removed (FIFO - First In, First Out).
